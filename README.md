### ČYZ app's backend server

----

Čyz - app designded to help Lithuania's police officers to mark citizens' property.

To try it, you need:
* Node.js >= 8.11.1
* MongoDB

**Simply run these commands:**

```bash

npm install --only=dev
npm run dev

```

----
Built with: Node.js, Express.js, MongoDB
